# React homework 7

1. Create a new branch in your repository for the homework 7
2. Install redux and redaxjs-toolkit to your app. Refactor all your existing code to use redux for managing state of an application and work with data.
3. No strict rules for folder structure for redux.
4. Implement the Order page. Route for Order page should be private. It means, that it should be available only for logged users.
5. The button "Place order" on the Home page should lead to the Order page.
6. **Open merge request**. Merge request link add to teams. After getting an approval of your MR from a mentor merge it to the main branch
   6.1 Merge request should include:
   - homework number
   - what was done (description of your changes)
   - screenshot of an implemented page.
- Please pay attention that for each homework you shell create separate branch.

You can merge your code only after getting an approval from a mentor and resolving all the conflicts if they exist.

Each pull request (PR) should contain only one commit. If there are several commits do not forget to make squash.
